# 🤖 AI Learning Buddy

An AI-powered educational assistant built using **Python, Streamlit, and Google Gemini AI**. The application helps students learn more effectively by providing explanations, summaries, quizzes, interview questions, real-life examples, and more through an interactive and user-friendly interface.

---

## 📖 Project Overview

AI Learning Buddy is designed to make learning simple, engaging, and personalized. Users can enter any topic, select a learning mode and difficulty level, and instantly receive AI-generated educational content.

This project demonstrates the integration of **Google Gemini AI** with **Streamlit** to create an intelligent learning assistant.

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/Shreya955177/AI_Learning_Buddy.git
```

### 2. Move into the project folder

```bash
cd AI_Learning_Buddy
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add your Gemini API Key

Create a **.streamlit/secrets.toml** file and add:

```toml
GEMINI_API_KEY="YOUR_API_KEY"
```

Or replace the API key directly in the code (not recommended for public repositories).

### 5. Run the application

```bash
streamlit run app.py
```

## 💡 How to Use

1. Open the application.
2. Enter a learning topic.
3. Select the difficulty level.
4. Choose a learning mode.
5. Click **Generate**.
6. View the AI-generated response.
7. Download the generated notes if required.
